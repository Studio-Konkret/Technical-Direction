## Hemuppgift
Ni ska sammanställa en HDA som genererar en procedurell geometri samt applicerar en procedurell textur via COPs. Ni kan återanvända en gammal procedurell setup, t.ex. staketet eller trälådan och bygga in ett procedurellt texturerings workflow. HDA:n behöver promotade parametrar för både genereringen av geometrin, samt för textureringen. Ett exempel på parameter för textureringen skulle kunna vara färgen, eller inställningar från Tile Pattern COP. Denna HDA ska ni hämta importera i Unreal Engine och visa att ni kan dra i parametrar i spelmotorn och dynamiskt ändra looken på er geometri.

Hållpunkter för att uppgiften fungerar så bra som möjligt.
- Kör Unreal 5.6 (tidigare versioner har inte stabilt stöd för COP-texturer)
- Installera Houdini Engine till Unreal på nytt via er nya Houdini 21.0 version (läs om installationen [här](https://github.com/Studio-Konkret/Technical-Direction/wiki/HDA-(Unreal-Engine)#installation))
    - Om det inte finns en plugin-mapp för er version så hade ni missat att klicka i Unreal Plugin i Houdini installationen. Om ni inte kan installera Houdini Engine för Unreal via Launchern i efterhand kan ni ladda ner plugin-mappen manuellt via denna [länk](https://drive.google.com/file/d/14kt6kVJ28A108mjct8YoXnNqieEK3AHL/view?usp=drive_link)
