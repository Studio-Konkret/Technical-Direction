
## Uppgifter


### Miniuppgift

Bekanta dig med VEX genom att skriva enkel kod som överför ett [fördefinierat attribut](https://www.sidefx.com/docs/houdini/unity/attributes.html#standard-houdini-attributes) till ett annat.

Ex:

`v@P` -> `v@Cd`

`v@N` -> `v@Cd`
